# 前端可视化拖拽平台

## 运行平台

- 安装相关依赖 npm i
- 运行后台代码
- yarn start

## 功能
- 登陆
- 注册
- 页面（组件）编辑
    - 复制组件
    - 删除组件
    - 组件组合，生成组件模版
    - 修改属性
- 代码预览
- 组件广场
    - 分为个人组件，组织组件，以及公共组件，根据不同的权限展示不同的组件
    - 预览组件
    - 编辑组件
- 组织管理
    - 申请组织
    - 管理申请列表，同意或拒绝
